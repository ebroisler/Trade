RESPOSTA DA QUESTÃO 2: 
NO MEU EXEMPLO, EU CRIEI UMA LISTA PARA ADICIONAR AS CATEGORIAS. CASO SEJA CRIADO UMA NOVA CATEGORIA, É NECESSÁRIO INSERI-LA NESTA LISTA DE CATEGORIAS. SE USARMOS INTEFACE, É SÓ CRIAR A NOVA PROPERTY DENTRO DA INTERFACE E IMPLEMENTAR SEU MÉTODO NAS CLASSES ONDE UTILIZA AS PROPRIEDADES DA INTERFACE.
ENTRETANTO, QUANDO USAMOS GERENCIADOS BANCO DE DADOS, A CATEGORIA DEVE SER CRIADO NELE PARA QUE OS DADOS FIQUEM PERSISTENTES E TRATAR ESTE NOVO CAMPO VIA CÓDIGO.
